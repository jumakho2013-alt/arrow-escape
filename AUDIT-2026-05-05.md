# Аудит Arrow Escape (Stray Arrows), 2026-05-05

## TL;DR
Игра в хорошем состоянии для релиза, ноль критических багов или утечек данных. Главные риски: (1) **AdMob ID в коде** — конкурент может перехватить трафик; (2) **handcrafted-levels.js 3.5MB загружается синхронно** — 200ms+ задержка старта на 3G; (3) **отсутствие CI/CD и README** — новый контрибьютор/будущий ты не сможет быстро пересобрать. Стоит починить эти три перед публикацией. Остальное — полировка, можно после релиза.

## Критические (must fix перед релизом)

### 1. AdMob ID в исходнике
- **Файл**: `index.html:1118-1123`
- **Суть**: 4 production AdMob unit IDs (iOS+Android, rewarded+interstitial) хардкод в JS. Конкурент может скопировать и направить трафик на свои объявления.
- **Фикс**: вынести в `capacitor.config.json` под `plugins.AdMob` или в build-time переменные. Либо хранить в Xcode/Gradle build configs.

### 2. Синхронная загрузка 3.5MB
- **Файл**: `index.html:74` (`<script src="handcrafted-levels.js?v=25">`)
- **Суть**: 461 уровень (3.5MB JS) парсится при старте, блокирует first-paint ~200ms на iPhone 12, ≥500ms на старых Android.
- **Фикс**: разбить на чанки по 50-100 уровней (`levels-1-100.js`, `levels-101-200.js`, ...), грузить ленивo по достижению. Или async load с прогресс-баром.

## Важные (should fix)

### 3. Empty catch blocks (~30)
- **Файлы**: `index.html:18, 85, 88, 147, 178, 274, 350, 384, 389, 436, ...`
- **Суть**: Тихие провалы. Если AdMob инит сломается, никто не узнает.
- **Фикс**: каждый `catch(e){}` заменить на `catch(e){console.error('контекст:', e)}` или вызов `_logAdErr()` где уместно.

### 4. Per-frame array allocations
- **Файл**: `index.html:2491-2495` (`drawArrowStatic`)
- **Суть**: `pts=[]` создаётся для каждой стрелки на каждом кадре. На лвл 100+ (60+ стрелок) ≈240 аллокаций/сек → GC pressure → подёргивания.
- **Фикс**: пул из reusable arrays или одна общая buffer-переменная, очистка между стрелками.

### 5. Отсутствует CSP
- **Файл**: `index.html` (`<head>`)
- **Фикс**: добавить `<meta http-equiv="Content-Security-Policy" content="default-src 'self' fonts.googleapis.com; script-src 'self'; style-src 'self' 'unsafe-inline'">`

### 6. Версия в 4 местах
- **Файлы**: `index.html` (`APP_VERSION`), `package.json`, `android/app/build.gradle` (`versionName`), `sw.js` (`CACHE_NAME`), плюс iOS Xcode.
- **Фикс**: один `version.json`, npm-скрипт `bump-version` который обновляет все.

### 7. README / DEV_SETUP отсутствует
- **Фикс**: создать `README.md` с: build команды, deploy в TestFlight/Play Console, архитектура (1 параграф), localStorage ключи, AdMob/реклама.

### 8. Magic numbers разбросаны
- **Файлы**: `index.html:487, 906, 1094-1097`
- **Суть**: `STARTING_COINS=150, HINT_COST=100, AD_COIN_REWARD=60, AUTO_ZOOM_TARGET_PX=14` константы — но рассыпаны по 50+ функциям.
- **Фикс**: единый `GAME_CONFIG = { STARTING_COINS: 150, HINT_COST: 100, ... }`.
